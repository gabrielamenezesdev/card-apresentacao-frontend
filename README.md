# card-apresentacao-frontend
Projeto de fundamentos web (HTML5 semântico, CSS3 e JavaScript) desenvolvido durante os meus estudos.

# 🚀 Card de Perfil Profissional

Este projeto foi desenvolvido para consolidar conhecimentos de base em desenvolvimento Web.

### 🛠️ Tecnologias Utilizadas:
- **HTML5:** Estrutura semântica (article, header, section, footer).
- **CSS3:** Box Model, Flexbox e estilização moderna.
- **JavaScript:** Manipulação de DOM e eventos de clique.

### 🧠 Aprendizados:
Durante a execução, resolvi bugs de carregamento de script e dominei a lógica de `box-sizing: border-box`.
